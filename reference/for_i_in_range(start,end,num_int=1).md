# for i in range(start,end,num: int=1)

> Adapted from [Python_Tutorial.pdf](../Python_Tutorial.pdf) — Chapter 3, LED Bar

A `for` loop is used to execute a program repeatedly, iterating in order over the items of a sequence (a list or a string).

- `start`: The initial value — the `for` loop starts with it.
- `end`: The ending value — the `for` loop ends with it.
- `num`: The amount automatically added to the value each time. The default value is `1`.
