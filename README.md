# Introduction to Python with Microcontrollers Course

This course uses the [Freenove Ultimate Starter kit for ESP32-S3 model FNK0082](https://freenove.com/fnk0082).  Much of the content is adapted from the Python_Tutorial.pdf included with the kit.  Some projects have been simplified, additional exploration options have been added.

## Use

To use this course clone the repo to your user folder and browse the projects via the github interface.

Run this command in the terminal to clone this repository:

```bash
git clone https://github.com/ColinKerr/python_microcontrollers.git
```


## How to contribute

1. Create an issue if you find an issue
1. Create a pull request to fix an issue
1. Suggest new projects or other changes via an issue

Be nice.
